## Materialize Dashboard

### Install

```
npm install
```

### Compile CSS

```
grunt sass
```

### Minify CSS

```
grunt cssmin
```

### Minify JS

```
grunt uglify
```

### Development watch

```
grunt dev
```

### Eslint testing

```
grunt ci
```

### Eslint fix

```
npm run fix
```

### Start server

```
npm start
```
